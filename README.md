# js-calendar

```
// 첨부 파일이 누락됐을 경우 깃으로 다운로드 가능합니다.
git clone https://github.com/bestseob93/js-calendar.git
```


### install with npm
```
npm install

// dev
npm start

// build
npm build
```

### install with yarn
```
yarn

// dev
yarn start

// build
yarn build
```

### build 할 경우

루트 폴더에 ./dist 경로에 파일이 생깁니다.
